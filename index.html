⚙️ 구글 시트 연동 설정

아래 단계를 따라 설정하면 구글 시트를 수정하는 것만으로
이 웹앱이 자동으로 업데이트됩니다.

1

구글 시트 열기

기존 카풀 스프레드시트를 열어주세요.
시트 이름이 당비비 카풀 일정 인지 확인하세요.

2

Apps Script 열기

상단 메뉴 → 확장 프로그램 → Apps Script 클릭

3

아래 코드를 전체 복사해서 붙여넣기

기존 내용을 모두 지우고 아래 코드로 교체하세요.

function doGet(e) { const sheet = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName('당비비 카풀 일정'); const rows = sheet.getDataRange().getValues(); // 헤더 2줄 건너뛰고 데이터만 추출 const data = []; for (let i = 2; i < rows.length; i++) { const row = rows[i]; if (!row[0]) continue; const date = Utilities.formatDate( new Date(row[0]), 'Asia/Seoul', 'yyyy-MM-dd'); data.push([ date, String(row[1] || ''), String(row[2] || ''), String(row[3] || '-').split('\n')[0], String(row[4] || '-').split('\n')[0], String(row[5] || '-'), String(row[6] || '-'), String(row[7] || '') ]); } const output = ContentService .createTextOutput(JSON.stringify({ ok: true, data: data })) .setMimeType(ContentService.MimeType.JSON); return output; }
4

배포하기

우측 상단 배포 → 새 배포 → 유형: 웹 앱
액세스 권한: 모든 사용자 → 배포 클릭

5

URL 복사 후 위에 붙여넣기

배포 완료 후 표시되는 웹 앱 URL을 복사해서
이 페이지 상단 입력창에 붙여넣고 저장 및 연결을 누르세요.
